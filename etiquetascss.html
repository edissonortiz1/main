¿Como funciona?

Al crear una hoja de estilos para un sitio web, CSS permite un trabajo más elaborado en la presentación, mayor rapidez a la hora de aplicar estos cambios, puesto que, si comparamos con HTML, allí se haría los cambios elemento por elemento y luego página por página, contrario a lo que pasa con CSS que al hacer cambios en la hoja de estilos automáticamente cambian para todas las páginas que contengan este estilo.


¿Cómo se conecta o interactúa con HTML?

HTML, tiene instrucciones donde se cargan las declaraciones de las hojas de CSS. Al abrir el portal del sitio web, este va a cargar tanto el código HTML como el CSS.

¿Qué nos ofrece CSS?

Mejor y mayor calidad en la presentación de fondos, textos, imágenes, colores y formas dentro de la misma página web. También el uso de elementos de multimedia está incluido en esta tecnología.

CSS, ha logrado desde 1996 iniciar una carrera, que hoy con el CSS3 casi la tiene ganada, en el sentido de generar una estandarización para la homogenización de la lectura del código de una web, por cada uno de los navegadores. Esto se da porque cada navegador interpreta diferente las pagina web y también porque las páginas web tienen muchas variables de construcción, lo que generaba errores o tal vez que la pagina no se presentara en forma óptima al usuario final.

Solo queda disfrutar el desarrollo de sitios o páginas web con mayor interactividad.
