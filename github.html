Para introducirnos en el tema de GIT y GITHUB, se debe tener en cuenta lo siguiente:

¿Qué es control de versiones?

Se define como control de versiones a la gestión de los diversos cambios que se realizan sobre los elementos de algún producto o una configuración del mismo es decir a la gestión de los diversos cambios que se realizan sobre los elementos de algún producto o una configuración

¿Qué es un repositorios?

Lugar donde se almacenan paquetes de software (Librerias, imágenes, código fuente, etc). Por lo regular los repositorios son servidores FTP o http.

¿Qué es GIT?

Es un software diseña por Linus Torvalds con el fin de realizar control de versiones a un producto en especifico.

¿Qué es GITHUB?

Es una herramienta que permite el desarrollo de software colaborativo, con el fin de alojar proyectos utilizando el control de cambio de GIT

¿Para que sirve GITHUB?

Aloja tu repositorio de codigo y te brinda herramientas para el trabajo en equipo dentro de un proyecto.

¿Como usar Github?

1- Se debe configurar un cuenta de Githut ingresado al portal oficial de ellos www.github.com, una vez en esta pagina se debe dirigir a sing in, ingresar un usuario, un correo valido y una contraseña.

2- Una vez ingresado los datos basicos, se debe el plan que se debe adquiri, por lo cual se escogera la cuenta gratis “Unlimited public repositories free”

3- Indicar las caracteristicas de tu perfil o experiencia, si es estudiante, si es un proyacto para la escual, etc.

4- Descargar el software de la pagina Github e instalar.

5- Una vez instaladado, se debe iniciar sesión con el usuario y contraseña creado
