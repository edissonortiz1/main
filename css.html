¿Qué es CSS?

(Cascading Style Sheets o CSS) Las hojas de estilo en cascada, son las reglas y estilos de representación en diferentes dispositivos, ya sean pantallas de equipos de escritorio, portatiles, moviles, impresoras u otro dispositivo capas de mostras contenidos web. En pocas palabras, sirve para ajustar el contenido de la pagina en el dispositivo que se observe y funciona en conjunto con HTML.

¿Qué es CSS3?

Cascading Style Sheets Level 3, como herramienta nos permite tener una gran variedad opciones para el diseño web y esta funcionando desde el 2005 y es el estandar actual.

Ventajas de trabajar en CSS3

• Es gratuito

• Todos los estilos son en una sola hoja

• Genera sitios más rapidos

• El diseño es adaptativo entre dispositivos

Modelo de cajas

Para le trabajo de las cajas, se debe tener en cuenta dos propiedades para definirlas, el ancho (height) y el alto (width).

Un ejemplo podria ser el siguiente

div{

background: #5DEAA8;

height: 100px; width: 200px;

width: 200px;

}

Adicionalmente a estas, tenemos el Margin, Padding y Border que nos brindaran mayor control al momento de organizar los elementos.

div{

border: 10px solid #3EC483;

background: #5DEAA8;

height: 100px;

width: 200px;

padding: 10px;

margin: 10px;

}

Zonas de elementos

Palabras clave para hacer referencia a una zona u orientación concreta sobre un elemento

• Top: Se refiere a la parte superior del elemento.

• Left: Se refiere a la parte izquierda del elemento.

• Right: Se refiere a la parte derecha del elemento.

• Bottom: Se refiere a la parte inferior del elemento.

• Center: En algunos casos se puede especificar el valor center para referirse a la posición central entre dos extremos.

Estilos internos y externos

• Internos: Se realiza utilizando un elemnto “style” y esta escrito en la parte inicial de HTML.

< !DOCTYPE html>

< html lang="es-ES">

< head>

< meta charset="utf-8">

< title>Ejemplo de estilo interno

< style>

p {color:red;}

< body>

< p>Esto es un párrafo de color rojo.< /p>

• Externos: Emplea un archivo CSS externo al documento HTML, usado por medio de un link el cual permite enlazar el HTML con un archivo externo.

< !DOCTYPE html>

< html lang="es-ES">

< head>

< meta charset="utf-8">

< title>Ejemplo de estilo interno< /title>

link rel="stylesheet" href="styles.css">

< body>

< p>Esto es un párrafo de color rojo.< /p>

Selectores CSS

• Selector universal: Se utilizan para seleccionar todos los elementos de la pagina, se indica mediante un (*).

• Selector de tipo o etiqueta: Selecciona todos los elementos de la pagina cuyaetiqueta HTML. Si se quiere aplicar los mismos estilos a dos etiquetas diferentes, se puede encadenar los selectores y permite agrupar todas las reglas individuales en una sola regla selector múltiple.

h1, h2, h3 {

color: #8A8E27;

font-weight: normal;

font-family: Arial, Helvetica, sans-serif;

}

• Selector descendente: Selecciona los elementos que se encuentran dentro de otros elementos. Se utiliza por medio del simbolo < p >. Los selectores descendentes permiten aumentar la precisión del selector de tipo o etiqueta. Así, utilizando el selector descendente es posible aplicar diferentes estilos a los elementos del mismo tipo. El siguiente ejemplo amplía el anterior y muestra de color azul todo el texto de los < span > contenidos dentro de un < h1 >

p span { color: red; }

h1 span { color: blue; }

• Selectores de ID: Permite seleccionar un elemento de la página a través del valor de su atributo id, solo selecciona un elemento de la pagina porque el valor del atributo id no se pueda repetir en dos elementos diferentes de una misma pagina. La sintaxis de los selectores de ID es muy parecida a la de los selectores de clase, salvo que se utiliza el símbolo de la almohadilla (#) en vez del punto (.) como prefijo del nombre de la regla CSS11
